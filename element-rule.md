## 编号规定
```
   4------------7  
  /|           /|  
 / |          / |     Z
5------------6  |     |
|  |         |  |     |
|  0---------|--3     O-------Y
| /          | /     /  
|/           |/     /
1------------2     X
```

**单元邻接点顺序**：{0, 1, 2, 3, 4, 5, 6, 7}  
**单元邻接单元顺序**：{x+, x-, y+, y-, z+, z-}  
**邻接单元起始点**：{{x+, 1}, {x-, 7}, {y+, 3}, {y-, 5}, {z+, 4}, {z-, 2}}
